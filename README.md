# AntiFraud
Проект для второго отборочного этапа олимпиады по промышленному программированию **PROD**.
Внутри используются Python 3.13 и PostgreSQL.
### Среди основных библиотек:
- FastApi
- Dishka
- SQLAlchemy
- Adaptix
- Descanso
- AioHTTP
- Pytest

> [!NOTE]
> На данный момент проект находится в том же состоянии, в котором и был сдан.
> Позже будут внесены несколько изменений, чтобы привести некоторые вещи в более подобающее состояние.
> Оригинальная версия проекта всегда будет находиться в ветке `original-state`.

> [!CAUTION]
> Возможно наличие говнокода и нарушение некоторых принципов чистой архитектуры
